Visibility Parameters

Tabs automatically adds paramets to the end of you url, like the sample below:

?tab=1

When tabs is present on the page, other micros can use this parameter as a visibility option. All micros have a visibility attribute at the settings level. In here you can set your conditional logic to display the micro if certain parameters are met.

The code snippit above will display the micro if the url parameter is '1' or if no parameter is present.

params[:tab].present? == false || params[:tab] == "1"

This code snippit will display the micros if the url parameter is '4'

params[:tab] == "4"

The visibility attribute is very flexible and can be used with an array of values. Check out the visibility page for more information
